# AppIndicator/KStatusNotifierItem Improved support for GNOME Shell
This extension integrates Ubuntu AppIndicators and KStatusNotifierItems (KDE's blessed successor of the systray) into GNOME Shell. Including support for legacy tray icons.
This extension supports left-click the icon to show the app itself if supported.
# 中文: 该扩展支持比KDE Plasma托盘更高级的功能: 左键点击托盘图标直接显示应用本身,若无法显示则显示托盘菜单,且该功能可以关闭(默认就是关闭的)
